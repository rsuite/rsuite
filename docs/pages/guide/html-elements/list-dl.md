### `<dl>`：HTML 描述列表元素

是一个包含术语定义以及描述的列表，通常用于展示词汇表或者元数据 (键-值对列表)。

<!--start-code-->

```js
const instance = (
  <dl>
    <dt>名称:</dt>
    <dd>React Suite</dd>
    <dt>出生年份:</dt>
    <dd>2016</dd>
    <dt>出生地:</dt>
    <dd>中国</dd>
    <dt>颜色:</dt>
    <dd>蓝色</dd>
  </dl>
);
ReactDOM.render(instance);
```

<!--end-code-->

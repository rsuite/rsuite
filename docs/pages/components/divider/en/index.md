## Props

### `<Divider>`

| Property    | Type`(default)`      | Description                           |
| ----------- | -------------------- | ------------------------------------- |
| children    | React.Node           | Set text in the dividing line         |
| classPrefix | string `('divider')` | The prefix of the component CSS class |
| vertical    | boolean              | Vertical dividing line                |

# Divider 分割线

`<Divider>` 分割线

## 获取组件

```js
import { Divider } from 'rsuite';
```

## Props

### `<Divider>`

| 属性名称    | 类型`(默认值)`       | 描述               |
| ----------- | -------------------- | ------------------ |
| children    | React.Node           | 设置分割线上的文本 |
| classPrefix | string `('divider')` | 组件 CSS 类的前缀  |
| vertical    | boolean              | 垂直分割线         |

# Divider 分割线

分隔线用于将内容水平或垂直分组。

## 获取组件

```js
import { Divider } from 'rsuite';

// or
import Divider from 'rsuite/lib/Divider';
```

## Props

### `<Divider>`

| 属性名称    | 类型`(默认值)`       | 描述               |
| ----------- | -------------------- | ------------------ |
| children    | ReactNode            | 设置分割线上的文本 |
| classPrefix | string `('divider')` | 组件 CSS 类的前缀  |
| vertical    | boolean              | 垂直分割线         |

### 多行输入框

<!--start-code-->
```js
const instance = (
 <div>
  <Input
    componentClass="textarea"
    rows={3}
    style={{ width:300 }}
    placeholder="Textarea"
  />
  <hr />
  <Input
    componentClass="textarea"
    rows={3}
    style={{ width:300, resize: 'auto' }}
    placeholder="resize: 'auto'"
  />
 </div>
);
ReactDOM.render(instance);
```

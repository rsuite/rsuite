# 基础样式

当使用了 React Suite 后，会重置一些 HTML 元素的样式。您可以直接使用 HTML 元素进行排版，展示标题、段落、列表、链接等等。

> 如果不需要这些样式，可以[配置不引入这些样式][config-reset-import]。

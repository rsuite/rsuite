### `<h1>`–`<h6>`：HTML 区域标题元素

下面的代码展示了所有可用的标题级别:

<!--start-code-->

```js
const instance = (
  <div>
    <h1>一级标题</h1>
    <h2>二级标题</h2>
    <h3>三级标题</h3>
    <h4>四级标题</h4>
    <h5>五级标题</h5>
    <h6>六级标题</h6>
    <hr />
    <hgroup>
      <h1>主标题</h1>
      <h2>副标题</h2>
    </hgroup>
  </div>
);
ReactDOM.render(instance);
```

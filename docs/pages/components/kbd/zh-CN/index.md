## Props

### `<Kbd>`

| 属性        | 类型`(默认值)`                        | 描述                  |
| ----------- | ------------------------------------- | --------------------- |
| children    | ReactNode                             | 键盘提示符的内容。    |
| classPrefix | string `('kbd')`                      | 组件 CSS 类名的前缀。 |
| size        | 'lg' \| 'md' \| 'sm' \| 'xs' `('md')` | 键盘提示符的大小。    |

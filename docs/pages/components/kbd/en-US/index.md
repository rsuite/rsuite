## Props

### `<Kbd>`

| Property    | Type`(Default)`                       | Description                            |
| ----------- | ------------------------------------- | -------------------------------------- |
| children    | ReactNode                            | The content of the kbd.                |
| classPrefix | string `('kbd')`                      | The prefix of the component CSS class. |
| size        | 'lg' \| 'md' \| 'sm' \| 'xs' `('md')` | The size of the kbd.                   |

# Whisper 弹窗触发器

`Whisper` 是一个触发浮动元素显示的组件。它主要用于包装 `Tooltip`、`Popover` 等需要悬浮显示的组件，提供了统一的触发行为和定位功能。通过 `Whisper`，您可以更灵活地控制浮层的显示和隐藏，以及自定义触发事件。

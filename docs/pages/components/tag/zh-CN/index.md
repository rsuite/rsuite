## Props

### `<Tag>`

| 属性名称    | 类型`(默认值)`        | 描述                   |
| ----------- | --------------------- | ---------------------- |
| children \* | ReactNode             | 组件的内容             |
| classPrefix | string `('tag')`      | 组件 CSS 类的前缀      |
| closable    | boolean               |
| as          | ElementType `('div')` | 为组件自定义元素类型   |
| onClose     | (event) => void       | 点击关闭按钮的回调函数 |

### Radio 组 - 横向布局


<!--start-code-->
```js
const instance = (
  <FormGroup controlId='radioList'>
    <RadioGroup name="radioList" inline>
      <Radio value="A">Item A</Radio>
      <Radio value="B">Item B</Radio>
      <Radio value="C">Item C</Radio>
      <Radio value="D" disabled>Item D</Radio>
    </RadioGroup>
  </FormGroup>
);
ReactDOM.render(instance);
```

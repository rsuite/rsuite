# Container 容器布局

容器布局可以定义页面的主体框架。

- `<Container>` 容器，用于组织布局结构。
- `<Header>` 容器顶栏。
- `<Content>` 内容部分。
- `<Footer>` 底部。
- `<Sidebar>` 侧边栏。

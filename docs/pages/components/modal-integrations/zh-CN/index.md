# Modal 集成 🧩

Modal 和 Drawer 与第三方库的集成。

## Nice Modal

[Nice Modal](https://github.com/eBay/nice-modal-react) 与 React Suite 的 Modal/Drawer 集成提供了轻量级、灵活定制的模态窗口解决方案。

### 安装

```bash
npm install @ebay/nice-modal-react
```

# Nextjs-CSS-Modules example

## How to use

Download the example:

```
curl https://codeload.github.com/rsuite/rsuite/tar.gz/main | tar -xz --strip=2 rsuite-main/examples/with-nextjs-css-modules
cd with-nextjs-css-modules
```

Install it and run:

```
npm install
npm run dev
# or
yarn
yarn dev
```
